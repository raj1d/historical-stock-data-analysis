# Historical Stock Data Analysis

## Project Overview
This project analyzes historical stock price and revenue data for Tesla (TSLA) and GameStop (GME) using Python. The objective is to collect, clean, analyze, and visualize financial data to identify trends and insights.

## Technologies Used
- Python
- Pandas
- yfinance
- BeautifulSoup
- Requests
- Matplotlib
- Jupyter Notebook

## Project Tasks
- Extracted historical stock data using the yfinance library.
- Collected revenue data through web scraping with BeautifulSoup.
- Cleaned and transformed datasets using Pandas.
- Visualized stock prices and revenue trends using Matplotlib.
- Compared historical performance of Tesla and GameStop.
